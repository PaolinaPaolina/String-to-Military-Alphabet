Install and Methods

Usage

First, include the class in your project.

<?php 
require 'militaryAlphaClass.php';
?>

Next, create a new instance of the class (you can name it what you wish, just change variable name):

<?php 
$mpa = new StringToMilitaryPhonetic();
?>

And that's really it. You now have access to all of the available functions.

Methods

Below you find a list of methods and how you can use them.

Convert
<?php
require 'militaryAlphaClass.php';
$mpa = new StringToMilitaryPhonetic();
$converted = $mpa->convert("string to convert");
?>

Parameters

<?php
require 'militaryAlphaClass.php';
$mpa = new StringToMilitaryPhonetic();
//This will not convert numbers or spaces
$converted = $mpa->convert("string to convert", true, true);
?>

Below you find a list of parameters (options) and how you can use them.

Convert Numbers

Do you NOT want the numbers to be converted (i.e. 4 to Four)? This is defaulted to convert numbers.

<?php
require 'militaryAlphaClass.php';
$mpa = new StringToMilitaryPhonetic();
//This will not convert number but will convert spaces
$converted = $mpa->convert("string to convert", true);
?>
Convert Spaces

Do you NOT want spaces to be converted (i.e. " " to space)? This is defaulted to convert spaces.

<?php
require 'militaryAlphaClass.php';
$mpa = new StringToMilitaryPhonetic();
//This will convert numbers but not spaces
$converted = $mpa->convert("string to convert", false, true);
?>
